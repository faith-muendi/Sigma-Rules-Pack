# Sigma Rules Pack

## Overview
This repository contains a curated collection of **Sigma detection rules** designed to identify common attacker techniques across Windows and Linux environments.

The rules are organized by **operating system** and **MITRE ATT&CK tactics**, making them easy to adapt for SIEM platforms such as Splunk, Elastic, Sentinel, and Wazuh.

---

## Objectives
- Develop high-quality Sigma detection rules
- Align detections with MITRE ATT&CK techniques
- Reduce false positives through tuned logic
- Demonstrate detection engineering best practices
